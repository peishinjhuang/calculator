
# C#－Winform：計算機
此程式在按下運算鈕(加減乘除)及等於按鈕(enter)時，會將文字框裡的數字加至總數。

功能包括以下：
*   附小數點
*   加減乘除
*   刪除尾數
*   顯示已輸入之算式
